Admin Panel
Manage Series Create, edit, and delete measurement series

Series management functionality will be implemented here.

Features include:

  • Add new series with name, min/max values, and color
  • Edit existing series properties
  • Delete series (with confirmation)
  • Validate min/max ranges
Manage Measurements Add, edit, and delete individual measurements

Measurement management functionality will be implemented here.

Features include:

  • Add new measurements to existing series
  • Edit measurement values and timestamps
  • Delete measurements
  • Bulk import from CSV/JSON
  • Validate values against series ranges
